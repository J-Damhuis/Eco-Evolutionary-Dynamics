This folder contains four items:

van velzen&etienne 2013.pdf is the final published version of the paper
"The evolution and coexistence of generalist and specialist herbivores under between-plant competition",
Ellen van Velzen & Rampal Etienne (2013) Theoretical Ecology 6:87-98

"simulation code" contains the c++ code for the main simulation and the simulated PIPs.

"textfiles" contains the textfiles from which the figures were created.

"figures_herbivorespecialization.R" is the R file containing to code to generate the figures.
